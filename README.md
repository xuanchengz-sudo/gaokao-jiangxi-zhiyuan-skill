# gaokao-jiangxi-zhiyuan-skill

面向江西高考报考与志愿填报的信息整理 Codex skill。

它用于：

- 江西高考志愿规则核验
- 成绩、位次和批次口径整理
- 院校专业组冲稳保候选筛选
- 专业选择建议与风险核验
- Excel/Markdown 志愿表辅助生成

## Install

```powershell
python C:\Users\zxc\.codex\skills\.system\skill-installer\scripts\install-skill-from-github.py --repo <owner>/gaokao-jiangxi-zhiyuan-skill --path skills/gaokao-jiangxi-zhiyuan
```

## Usage

在 Codex 中使用：

```text
Use $gaokao-jiangxi-zhiyuan 帮我整理江西高考志愿冲稳保方案。
```

## Boundary

本 skill 只做信息整理，不输出录取概率、不承诺录取、不替考生做最终决策。所有当年分数线、志愿时间、招生计划、投档数据和专业目录均应以江西省教育考试院及院校官方发布为准。
