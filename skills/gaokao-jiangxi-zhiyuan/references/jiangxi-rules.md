# 江西规则核验清单

本文件只保存江西志愿工作的核验顺序和口径提醒。任何当前年份的具体时间、批次、志愿数、分数线、投档表、征集安排，都必须以江西省教育考试院当年公告为准。

## 江西口径要点

- 江西已进入新高考 `3+1+2` 口径：普通类按首选科目分为物理类、历史类。当前年份仍需官方核验。
- 普通类志愿一般按“院校专业组”组织，专业组内有专业调剂和退档风险；不要按“一个学校=一个志愿”解释。
- 同一院校可有多个专业组，不同组的选科要求、专业、学费、校区、培养方式可能完全不同。
- 艺术类、体育类、三校生、专项计划、定向、中外合作等有独立规则。遇到这些类别，先拆分口径，再分别核验批次和投档规则。
- 江西艺术类常涉及文化分、专业统考成绩、综合成绩、专业方向和省统考类别；不要把普通类位次直接套到艺术类。
- 专科普通类与本科普通类可使用相同的位次思路，但分数线、批次时间、计划和投档表必须单独取数。

## 每次回答前的 P0 检查

1. 确认年份：2026、2027 等，不要默认当前年数据已经入库。
2. 确认阶段：
   - 考前：规则科普、目标区间、模拟位次，只能估算。
   - 出分前：仍只能用往年一分一段估算。
   - 出分后填报期：优先用真实位次，必须附填报截止时间。
   - 录取期/征集期：必须查当轮征集公告和缺额计划。
3. 确认类别：普通物理、普通历史、艺术、体育、三校生、专项、定向等。
4. 确认数据完整性：一分一段、投档表、计划、选科要求、专业目录、章程缺任何一项都要标注。
5. 输出来源：每张表至少保留数据来源列或来源页。

## 不可写死的字段

以下字段每年可能变化，必须实时核验或从当年本地数据读取：

- 志愿填报时间、确认时间、征集时间。
- 各批次可填志愿数量和每个志愿可填专业数。
- 本科线、专科线、特殊类型控制线、艺术/体育文化线和专业线。
- 投档排序规则、同分排序规则、综合分公式。
- 专项计划、定向、地方专项、苏区专项的资格和批次。

## 输出措辞

- 可以说：“该字段待当年官方公告确认，暂不写死。”
