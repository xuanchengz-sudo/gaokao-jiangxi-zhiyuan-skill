# 来源验证与联网补数规则

用于判断网页爬取、搜索结果、转载数据和本地缓存文件是否可用于江西高考志愿表。核心原则：先官方、后线索；能验证才入主表，不能验证就标注待核验。

## 来源等级

- A0 官方原始来源：江西省教育考试院 `jxeea.cn`、江西省教育厅 `jyt.jiangxi.gov.cn`、江西普通高考综合管理平台 `jxgk.jxeea.cn`、教育部阳光高考 `gaokao.chsi.com.cn`、教育部/省级主管部门官网。
- A1 院校官方来源：院校本科/专科招生网、院校招生章程、院校艺术类招生简章。需能从院校官网或阳光高考院校库反查到该招生网入口。
- B 可验证转载/聚合来源：第三方网站、新闻页、公众号镜像、网盘资料包、截图 OCR、搜索引擎快照。只有在能找到官方原始 URL、官方附件名、发布时间、表头和关键行一致时，才可作为补充；否则只能作为检索线索。
- C 不可验证来源：论坛问答、SEO 汇总页、无出处截图、无发布日期/无表头的表格、无法定位原始来源的下载文件。不得作为硬数据。

## 检索顺序

1. 查本地结构化数据和已有缓存，确认字段、年份、批次、科类、来源 URL 是否齐全。
2. 查官方域名：
   - `site:jxeea.cn 江西省 {year} 普通高校招生 {批次} 投档情况统计表 {科类}`
   - `site:jxeea.cn 江西省 {year} 一分一段表`
   - `site:jxeea.cn 江西省 {year} 招生计划 艺术类 舞蹈类`
   - `site:gaokao.chsi.com.cn {院校名} 招生章程 {year}`
   - `site:{院校官网域名} {year} 招生章程 艺术类 舞蹈`
3. 若官方站内检索不到，进行全网搜索，把第三方结果当线索，回溯其中的官方链接、附件名、截图水印、发布时间和表头。
4. 若近年数据在官方综合平台登录后才可见，不绕过登录、不伪造请求、不抓取未授权接口；在输出中标注“需登录平台核验”。
5. 若只找到第三方转载，至少做两项交叉验证：与官方新闻标题/发布时间一致、与官方表头/批次/科类一致、与另一个独立来源关键行一致、文件元数据或截图水印指向官方。仍需标 B，并附复核说明。

## 可入表条件

每条硬数据至少满足：

- 年份、批次、科类/艺术子科类、院校专业组或专业粒度明确。
- 最低分、最低位次/投档排名、计划数等字段含义明确。
- 来源等级为 A0/A1；或 B 级但已回溯到 A 级原始依据并在备注写明。
- URL 可访问，或有本地缓存文件且记录原始 URL、缓存路径、文件大小/哈希、最后核验时间。

不满足以上条件时：

- 表格中写空值或“待核验”，不要补猜。
- 在“来源审计/数据状态”表记录检索词、找到的页面、未采用原因。
- 第三方数据最多进入“待复核线索”列，不参与冲稳保分档。

## 输出字段

联网检索或爬取后，候选表或来源表应尽量保留：

- `source_level`：A0/A1/B/C。
- `source_url`：实际取数 URL。
- `official_url`：可回溯的官方 URL；没有则留空。
- `retrieved_url`：搜索或爬取命中的 URL。
- `verification_status`：已官方核验/已交叉验证/待登录核验/不可验证。
- `verification_method`：站内检索、官方附件、院校章程、跨源比对、本地缓存等。
- `fetched_at`、`last_verified_at`。
- `cache_path`、`sha256`：使用本地文件时记录。
- `notes`：口径差异、不可混用原因、缺失字段。

## 江西艺术类特别规则

- 舞蹈类使用艺术子科类/统考类别、综合分、投档排名；不要用普通历史/物理位次替代。
- 2024 起新高考院校专业组口径可能与 2023 及以前不同；跨年比较必须标注口径差异。
- “专业分数线”若是普通历史/物理逐专业录取，不得混入艺术舞蹈类表。
- 同一院校有多个舞蹈类专业组时，保留专业组代码、投档线、最低投档排名明细，不只写学校最低值。

## 推荐结论措辞

- 可写：“已核到 A0 官方投档表，字段可入主表。”
- 可写：“找到第三方转载，但未回溯到官方原始表，列为待复核线索。”
- 可写：“2025 逐校明细需要登录江西普通高考综合管理平台，当前不填确数。”
- 不要写：“网上有这个数据，直接按它算。”
